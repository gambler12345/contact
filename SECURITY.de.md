## Sicherheitslücken melden

Wir nehmen Sicherheitsprobleme ernst. Wenn Sie eine Sicherheitslücke entdecken, öffnen Sie bitte **kein** öffentliches Issue.

Bitte melden Sie es stattdessen privat, indem Sie eine [GitHub Security Advisory](../../security/advisories/new) öffnen oder die Maintainer direkt über die in diesem Repository angegebenen Kontaktinformationen kontaktieren.

Beim Melden einer Sicherheitslücke geben Sie bitte folgendes an:
- Eine Beschreibung der Sicherheitslücke und ihrer potenziellen Auswirkungen
- Schritte zur Reproduktion oder ein Proof-of-Concept
- Alle vorgeschlagenen Abhilfemaßnahmen oder Korrekturen

Sie können innerhalb von 48 Stunden eine Bestätigung und einen Lösungszeitplan erwarten, sobald das Problem bewertet wurde. Wir bitten Sie, uns eine angemessene Zeit zu geben, das Problem zu beheben, bevor eine öffentliche Offenlegung erfolgt.
